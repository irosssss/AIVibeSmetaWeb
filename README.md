# AIVibe — веб-смета (отдельный проект)

Веб-кабинет дизайнера AIVibe со **сметой-комплектацией** (FF&E: мебель, свет, сантехника, декор).
Это **единственный дом** этого сервиса — работаем только здесь, чтобы не было копий и путаницы.

## Как запустить
- `npm install`, затем `npm run dev` → `http://localhost:5173` (или через **Claude Preview**:
  конфиг `.claude/launch.json`, имя `aivibe-web` — vite на порту 8753).
  Внизу страницы переключатель Промо / Кабинет / Админка.
- Продакшн: `npm run build` → статика в `dist/`; локальный просмотр — `npm run preview`.
- Тесты: `npm test` (vitest, `tests/*.test.js`).

## Структура
- `web/` — код сайта (React + Vite; компоненты исторически общаются через `window.*`,
  вход — `web/main.jsx`, мост npm→глобалы — `web/vendor-globals.js`):
  - `web/components/project-detail.jsx` — смета по комнатам (наценка, переключатель «Рабочая / Для клиента»);
  - `web/components/cabinet-views.jsx` — «Мои проекты», кнопка «Импорт из Excel»;
  - `web/engine.js` — движок сметы (эргономика + бюджет-оптимизатор);
  - `web/pdf.js` — выгрузка PDF; `web/xlsx.js` — выгрузка и импорт Excel;
  - `web/project-data.js` — данные проектов (в т.ч. реальная «Кирова»).
- `docs/` — планы, бэклог, аудиты, стратегия. **Начать с `docs/WEB_SMETA_ROADMAP.md`** (история функций + план).

## Что сделано (кратко)
Движок эргономики + бюджет-оптимизатор · смета по комнатам · наценка дизайнера (две цены) ·
выгрузка PDF и Excel · **два режима выгрузки** (Рабочая / Для клиента) · **импорт сметы из Excel**.
Подробно — в `docs/WEB_SMETA_ROADMAP.md`.

## Связь с основным проектом AIVibe
Этот проект **выделен из монорепо AIVibe** (iOS + backend + web). В монорепо остаются и не теряются:
- история коммитов веб-работы;
- backend-заглушка оплаты/входа — `backend/functions/web-billing/` (когда дойдём до реальной оплаты);
- iOS- и backend-документы.

Дальнейшую работу над веб-сметой ведём **в этом проекте**; копия `web/` в монорепо помечена как архив.
