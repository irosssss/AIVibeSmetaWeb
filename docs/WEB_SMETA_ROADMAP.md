# Веб-смета AIVibe — история функций и план расширения

> Единый журнал по веб-поверхности `web/` (кабинет дизайнера со сметой-комплектацией).
> Ведём по факту вех. Связанные документы: бэклог-приоритеты — `SMETA_BENCHMARK_2026-06.md`;
> стратегия — `STRATEGY_SYNTHESIS_2026-06-18.md`, `STRATEGY_SCENARIOS_2026-06-24.md`;
> бизнес-модель — `BUSINESS_MODEL.md` (пивот на фабрики, v1.2).

## 0. Что это и рамка

**Веб-кабинет дизайнера** — одна из трёх поверхностей AIVibe (`AIVibe/` iOS, `backend/` Node, `web/` веб).
Сейчас это приоритетный фокус (пивот: дизайнеры/стейджинг, web-first).

**Что мы строим:** **спецификацию/комплектацию (FF&E)** — мебель, свет, сантехника, декор.
В терминах дизайнеров это НЕ «смета» (работы + черновые материалы) и НЕ «ведомость» (отделка),
а именно комплектация. «Смета» оставляем как разговорное название (см. `SMETA_BENCHMARK §0`).

**Источник цен/SKU:** партнёрский каталог фабрик (фид в YDB), НЕ маркетплейсы и НЕ скрейпинг розницы.

**Стек:** фронт — React 18 без сборки (Babel-standalone, скрипты `web/components/*.jsx`);
PDF — pdfmake (CDN), Excel — SheetJS (CDN); бэк — Node 20 + YDB; AI — YandexGPT/GigaChat; оплата — ЮKassa.

**Где код и как запускать:** всё в `web/` (на ветке master; правки ведём в основной папке).
Запуск превью — `.claude/launch.json` (конфиг `aivibe-web`, статик-сервер на `web/`).
